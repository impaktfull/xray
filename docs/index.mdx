# xRay (inspector)

A comprehensive Flutter debugging toolkit that captures and displays network calls, logs, and key-value storage — all inspectable in real-time via a remote UI.

## What's included

| Package                             | Description                                                         |
| ----------------------------------- | ------------------------------------------------------------------- |
| `xray_inspector`                    | Core package — network, log, and key-value inspectors + HTTP server |
| `xray_network_dio_interceptor`      | Dio HTTP interceptor                                                |
| `xray_shared_preferences_inspector` | SharedPreferences key-value inspector                               |
| `xray_secure_storage_inspector`     | flutter_secure_storage inspector                                    |

## Architecture overview

```
Your Flutter app
       │
       ├── XRayNetworkInspector   ◄── XRayNetworkDioInterceptor
       ├── XRayLogInspector
       ├── XRaySharedPreferencesInspector
       └── XRaySecureStorageInspector
                        │
              XRayInspectorServer  (HTTP + WebSocket + UDP discovery)
                        │
                  xRay Inspector App / Web App
```

The inspector server runs inside your app and broadcasts data over HTTP/WebSocket. Connect to it using the xRay app or web app.

## Design principles

- **No singletons** — all dependencies are passed explicitly via constructors.
- **No disk I/O** — all captured data lives in memory only.
- **Composable** — add only the inspectors you need.
