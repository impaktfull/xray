# xray_inspector_ui

Flutter UI package that connects to a running `XRayInspectorServer` and displays network calls, log entries, and key-value data in real-time.

## Installation

```yaml
dependencies:
  xray_inspector_ui: ^0.0.1
  xray_inspector_core: ^0.0.1
```

## XRayInspectorUiController

The controller manages the connection to the inspector server and holds all received data.

```dart
import 'package:xray_inspector_ui/xray_inspector_ui.dart';

final controller = XRayInspectorUiController();
```

### Connecting

```dart
// Auto-discover on the local network via UDP broadcast
await controller.connectLocal();

// Connect to a specific discovered server
await controller.connect(discoveredServer);

// Disconnect
controller.disconnect();
```

### Reactive data

```dart
// Network calls keyed by inspector ID
controller.networkCalls   // ValueNotifier<Map<String, List<ImpaktfullNetworkCall>>>

// Log entries keyed by inspector ID
controller.logEntries     // ValueNotifier<Map<String, List<ImpaktfullLogInterceptorEntry>>>

// Key-value entries keyed by inspector ID
controller.keyValueEntries // ValueNotifier<Map<String, List<InspectorKeyValueEntry>>>
```

## Screens

### InspectorSettingsScreen

Entry point. Shows connection status and links to all inspectors.

```dart
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (_) => InspectorSettingsScreen(controller: controller),
  ),
);
```

### NetworkCallsScreen

List of all captured network calls for a given inspector.

```dart
NetworkCallsScreen(
  controller: controller,
  inspectorId: 'network',
)
```

### NetworkCallDetailScreen

Full detail view for a single network call — headers, body, status, duration.

### LogEntriesScreen

List of all log entries for a given inspector, filterable by level.

### LogEntryDetailScreen

Full detail view for a single log entry.

### KeyValueScreen

Browse all key-value entries for a source. Supports inline editing and deletion.

### KeyValueEntryDetailScreen

Full detail view for a single key-value entry.

## Widgets

### InspectorOverlayFab

Floating action button overlay for quick access to the inspector. Add it to the top of your widget tree:

```dart
Stack(
  children: [
    YourApp(),
    InspectorOverlayFab(controller: controller),
  ],
)
```

### NetworkStatusBadgeWidget

Visual badge showing the status of a network call (`loading`, `success`, `error`).

```dart
NetworkStatusBadgeWidget(status: call.status)
```

### NetworkMethodBadgeWidget

Coloured badge displaying the HTTP method.

```dart
NetworkMethodBadgeWidget(method: call.method)
```

### LogLevelBadgeWidget

Coloured badge displaying the log level.

```dart
LogLevelBadgeWidget(level: entry.level)
```

## Dependencies

This package depends on `impaktfull_ui` for its visual components.
