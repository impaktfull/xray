# impaktfull_network_dio_interceptor

Dio interceptor that captures all HTTP requests, responses, and errors into an `ImpaktfullNetworkInspector`.

## Installation

```yaml
dependencies:
  impaktfull_inspector: ^0.0.1
  impaktfull_network_dio_interceptor: ^0.0.1
  dio: ^5.0.0
```

## Usage

```dart
import 'package:dio/dio.dart';
import 'package:impaktfull_inspector/impaktfull_inspector.dart';
import 'package:impaktfull_network_dio_interceptor/impaktfull_network_dio_interceptor.dart';

final networkInspector = ImpaktfullNetworkInspector();

final dio = Dio()
  ..interceptors.add(
    ImpaktfullNetworkDioInterceptor(inspector: networkInspector),
  );
```

## How it works

`ImpaktfullNetworkDioInterceptor` hooks into Dio's interceptor lifecycle:

1. **`onRequest`** — Creates a new `ImpaktfullNetworkCall` with status `loading` and stores the call ID in Dio's request options `extra` map.
2. **`onResponse`** — Finds the call by ID and updates it with the response status code, headers, and body. Status becomes `success`.
3. **`onError`** — Finds the call by ID and updates it with the error details. Status becomes `error`.

## Lifecycle states

| State | When |
|---|---|
| `loading` | Request sent, waiting for response |
| `success` | Response received with any status code |
| `error` | Network error or Dio error |

## Combining with other Dio interceptors

Add `ImpaktfullNetworkDioInterceptor` alongside other interceptors. Order matters — place it where you want the captured request/response to reflect the transformation state:

```dart
final dio = Dio()
  ..interceptors.addAll([
    AuthInterceptor(),                                        // runs first
    ImpaktfullNetworkDioInterceptor(inspector: inspector),   // captures after auth headers added
    LogInterceptor(),
  ]);
```
