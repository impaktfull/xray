# xray_network_serverpod_interceptor

HTTP client wrapper compatible with Serverpod that captures all network calls into an `XRayNetworkInspector`.

## Installation

```yaml
dependencies:
  xray_inspector: ^0.0.1
  xray_network_serverpod_interceptor: ^0.0.1
```

## Usage

```dart
import 'package:xray_inspector/xray_inspector.dart';
import 'package:xray_network_serverpod_interceptor/xray_network_serverpod_interceptor.dart';

final networkInspector = XRayNetworkInspector();

final httpClient = ImpaktfullServerpodHttpClient(inspector: networkInspector);
```

Pass the `httpClient` to your Serverpod client wherever it accepts a custom HTTP client.

## How it works

`ImpaktfullServerpodHttpClient` wraps the underlying HTTP client and intercepts all outgoing requests and incoming responses, forwarding the data to the `XRayNetworkInspector` in the same way as the Dio interceptor.
