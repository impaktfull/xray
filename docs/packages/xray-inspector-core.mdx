# xray_inspector_core

Pure Dart package (no Flutter dependency) containing the shared data models and protocol constants used across the xray_inspector ecosystem.

## Installation

```yaml
dependencies:
  xray_inspector_core: ^0.0.1
```

You typically don't need to add this package directly — it's pulled in transitively by `xray_inspector` and `xray_inspector_ui`.

## Models

### ImpaktfullNetworkCall

Immutable model representing a single captured HTTP call.

```dart
class ImpaktfullNetworkCall {
  final String id;                          // Unique ID
  final String method;                      // HTTP method
  final String uri;                         // Full URI
  final Map<String, dynamic>? requestHeaders;
  final dynamic requestBody;
  final int? responseStatusCode;
  final Map<String, dynamic>? responseHeaders;
  final dynamic responseBody;
  final ImpaktfullNetworkCallStatus status; // loading / success / error
  final DateTime startTime;
  final DateTime? endTime;
  final String? error;
  int? get durationMs;                      // Computed from start/end
}
```

### ImpaktfullNetworkCallStatus

```dart
enum ImpaktfullNetworkCallStatus {
  loading,
  success,
  error,
}
```
