# Guides

Deep-dive guides for each inspector feature. Use these after the [Quick Start](/docs/getting-started) to get the most out of each component.

## Available guides

### [Network Inspection](/docs/guides/network-inspection)

Capture and inspect HTTP requests and responses in real time. Covers setup with Dio and Serverpod, accessing captured calls, and the full network call model.

### [Log Inspection](/docs/guides/log-inspection)

Record and view structured log entries with severity levels (verbose, debug, info, warning, error). Covers setup, writing logs, and filtering.

### [Key-Value Inspection](/docs/guides/key-value-inspection)

Inspect, edit, and delete SharedPreferences and flutter_secure_storage entries at runtime. Covers setup for both storage backends.

### [Remote UI](/docs/guides/remote-ui)

Connect a separate Flutter UI to your running inspector server over the local network. Covers automatic UDP discovery, manual connection, and the `XRayInspectorUiController`.
